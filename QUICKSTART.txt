## 🎯 STEP 1: Install Node.js (5 Minutes)

### Option A: Download & Run Installer (Easiest)
1. Go to: https://nodejs.org
2. Click the big **LTS** button (Left side)
3. Wait for download
4. Double-click the file and run the installer
5. Click "Next" through everything
6. Click "Finish"
7. **Close and reopen PowerShell**

### Option B: Via Chocolatey (If You Have It)
```powershell
choco install nodejs
```

### Verify Installation
```powershell
node --version
npm --version
```

Both should show version numbers like `v20.11.0` and `10.5.0`

---

## 🎯 STEP 2: Copy-Paste These Commands

Open PowerShell and run these commands one by one:

```powershell
cd C:\Users\Administrator\Downloads\sparkhome
```

Wait for prompt, then:

```powershell
npm install
```

Wait for it to finish (2-5 minutes), then:

```powershell
npm run build
```

Wait for `✓ build complete`, then:

```powershell
npm install -g vercel
```

Then:

```powershell
vercel login
```

Then:

```powershell
vercel --prod
```
